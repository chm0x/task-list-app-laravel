# TASK LIST APP

Description:

A basic Task List App that you can save on a Database, list aa task and mark completed tasks. 

Tools:
* Laravel 11
    * Route and Blade Only Basics. 
* MySQL 8
* A little TailwindCss


Installation: 
1. 
    ```
    > composer install
    ```
2. 
    ```
    > npm install
    > npm run dev
    ```
